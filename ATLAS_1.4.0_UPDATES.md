# Atlas 1.4.0 updates

- Added note-name typing for frequency controls, so values like C4, D#2, or Bb1 can be entered directly.
- Added the accessibility option to read frequency controls in Hz instead of semitones.
- Improved modulation assignment: add modulation is now a submenu, existing modulations can be edited or removed from the parameter menu, and modulation peak values can be typed as real destination values.
- After setting a modulation amount, focus now returns to the parameter where the modulation was added instead of jumping back to global controls.
- Improved effect reordering so the visible effect groups follow the actual chain order, with keyboard shortcuts for moving modules earlier or later.
- Added 10x rate options for LFOs and Random LFOs.
- Split Random LFOs into Random 1, Random 2, Random 3, and Random 4 groups for easier VoiceOver navigation.
- Cleaned up Random LFO accessible labels, including clearer tempo sync and transport sync controls.
